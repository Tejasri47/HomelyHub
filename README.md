#HomelyHub A web app to find and book homestays with a clean, homely experience.
#Features:
Browse properties with photos, price, and amenitiesFilter by location, dates, guests, and price rangeUser auth for hosts and guestsBooking flow with calendar availabilityHost dashboard to list and manage properties
#Tech Stack
#Frontend: React, Redux, Tailwind CSS
#Backend: Node.js, Express
#Database: MongoDB + MongooseAuth: JWT
